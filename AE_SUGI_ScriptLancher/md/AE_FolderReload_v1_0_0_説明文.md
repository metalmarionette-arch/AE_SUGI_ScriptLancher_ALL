# フォルダ再読み込み 説明文（概要・使い方）

## 機能概要
このスクリプトは、After Effects で **選択フォルダ配下のフッテージを再読込し、更新済みソースを反映します。** ためのツールです。

## 使い方
1. 再読込したい素材フォルダを選択します。
2. 必要に応じて対象フィルタを設定します。
3. 実行してリンク素材を再読み込みします。
4. 置換結果や欠落リンクを確認します。

## 主な活用シーン
- 外部ツールで上書きされた素材の反映
- レンダー差し替えの一括更新
- 更新漏れチェックの効率化

## 注意点
- 実行前にプロジェクトを保存しておくと安全です。  
- 複数レイヤーを対象にする場合、選択順や選択範囲で結果が変わることがあります。  
- 意図しない結果になった場合は `Undo` で戻し、設定を見直して再実行してください。
