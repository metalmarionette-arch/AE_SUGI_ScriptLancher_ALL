# 位置統合（Unsplit） 説明文（概要・使い方）

## 機能概要
このスクリプトは、After Effects で **分離次元のPositionを統合し、単一Positionへ戻します。** ためのツールです。

## 使い方
1. 分離次元を解除したいレイヤーを選択します。
2. 実行してPosition次元を統合します。
3. 既存キーの変換結果を確認します。
4. 必要なら再度Splitして編集方針を調整します。

## 主な活用シーン
- 受け取ったデータの編集方式統一
- 式運用で単一Positionが必要な場合
- 軸別編集完了後の整理

## 注意点
- 実行前にプロジェクトを保存しておくと安全です。  
- 複数レイヤーを対象にする場合、選択順や選択範囲で結果が変わることがあります。  
- 意図しない結果になった場合は `Undo` で戻し、設定を見直して再実行してください。
