# シーケンサーオフセット 説明文（概要・使い方）

## 機能概要
このスクリプトは、After Effects で **複数レイヤーの開始タイミングを一定間隔でずらして配置します。** ためのツールです。

## 使い方
1. 対象レイヤーを選択します。
2. オフセット間隔（フレーム/秒）と並び順を設定します。
3. 基準点（先頭/現在時間など）を選んで実行します。
4. 並び順と重なりをタイムラインで確認します。

## 主な活用シーン
- 連番演出やドミノ表示
- 文字/アイコンの段階出し
- 手作業での時間配置を削減

## 注意点
- 実行前にプロジェクトを保存しておくと安全です。  
- 複数レイヤーを対象にする場合、選択順や選択範囲で結果が変わることがあります。  
- 意図しない結果になった場合は `Undo` で戻し、設定を見直して再実行してください。
