# 末尾処理付きプリコンポーズ 説明文（概要・使い方）

## 機能概要
このスクリプトは、After Effects で **プリコンポーズ時の命名や末尾処理を統一し、作業手順を標準化します。** ためのツールです。

## 使い方
1. プリコンポーズしたいレイヤーを選択します。
2. 生成名や末尾文字列のルールを設定します。
3. 実行してプリコンポ化します。
4. 元コンポと生成先の構造を確認します。

## 主な活用シーン
- 命名ルール付きの量産作業
- 手作業での名称ゆれ防止
- チーム運用時の統一フォーマット化

## 注意点
- 実行前にプロジェクトを保存しておくと安全です。  
- 複数レイヤーを対象にする場合、選択順や選択範囲で結果が変わることがあります。  
- 意図しない結果になった場合は `Undo` で戻し、設定を見直して再実行してください。
