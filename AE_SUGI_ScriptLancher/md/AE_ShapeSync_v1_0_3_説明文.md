# シェイプ同期 説明文（概要・使い方）

## 機能概要
このスクリプトは、After Effects で **シェイプレイヤー間でパスやスタイル情報を同期し、見た目を統一します。** ためのツールです。

## 使い方
1. 同期元と同期先のシェイプレイヤーを選択します。
2. 同期対象（パス/塗り/線/トランスフォーム）を指定します。
3. 実行して設定を反映します。
4. 破綻がないか各グループを確認します。

## 主な活用シーン
- 複数カットで同一デザイン維持
- 修正反映の横展開
- 手動コピペ時の抜け漏れ防止

## 注意点
- 実行前にプロジェクトを保存しておくと安全です。  
- 複数レイヤーを対象にする場合、選択順や選択範囲で結果が変わることがあります。  
- 意図しない結果になった場合は `Undo` で戻し、設定を見直して再実行してください。
